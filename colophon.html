I now do all of my writing in iA Writer by Information Architects. It’s a blank white screen with one font in one size and one color and it’s absolutely perfect. That it pre-formats Markdown makes it the perfect tool for no-fuss, distraction-free writing.

I now do all of my coding in Sublime Text 2, as muscular a text editor as I’ve ever seen.

This site and is built on the Ruby on Rails framework, and hosted on Heroku.

All text is set in Charis SIL by SIL International.

Colors borrowed from the Solarized color palette by Ethan Schoonover.

Icons are from a great icon font called Modern Pictograms by The Design Office.
